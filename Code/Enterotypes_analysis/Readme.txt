In the Calculating_ISN_per_Enterotype\ folder, there are the early stages of the Enterotype analysis, i.e., the calculation of the ISNs that are the input for the analysis. 
The analysis in this folder, on the contrary are post-processing analysis after the Encoding/Decoding pipeline (PLEX.I) has ben ran. We refer to Plex.I paper ( https://pubmed.ncbi.nlm.nih.gov/37928248/ ) for details on the Encoding/decoding procedures. 

To help the reader understand the various steps, we generate a markdown file showcasing the intermediate and final results.
